An agent YAML file is a configuration file used to declaratively define the behavior, memory, tools, and background instructions for AI agents
- The AgentSchema specification is an open standard for defining AI agents
- azd ai agent init -m command reads it. asks user for any var value if not specified
- azd deploy or azd ai agent run
